How Boundary Works

  •  
    Identity-based Access
    Enable privileged sessions for users and applications based on user identity and role.
  •  
    Service Discovery
    Service discovery focuses on automating the process of onboarding new or changed infrastructure resources to Boundary as hosts.
  •  
    Infrastructure as Code with Terraform
    Define policies and manage Boundary with Terraform provider that supports the full breadth of Boundary configurations.
  •  
    Session Visibility and Audit Logs
    Visibility into session metrics, events, logs, and traces with the ability to export data to business intelligence and event monitoring tools
